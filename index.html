ABOUT ME

I’m a highly innovative software developer with a passion for integrating the art world with the digital world, and everything in between. Check out some of my recent projects.

Prior to becoming a software engineer, I was managing and operating a father-son family business that specialized in manufacturing. During that period I learned how to manage people, production schedules, inventories and everything else that comes.

When I'm not coding I'm most likely pursuing another type of creative endeavor whether it be screen printing, print-making, illustrating, photography, drawing or painting, I'm always making something!

Projects

PROJECTS

odaat
Live | Github
One Day At A Time is an original social media app that is catered towards members of Alcoholics Anonymous. Users can host or browse meetings, official and non-official, through a Google Maps integrated API. It also has the functionality of other traditional social media apps.
React / Redux / Javascript / Node.js / Python / Flask / SQLAlchemy / JSX / CSS / AWS3 / Google Maps API Docker
Money Pit
Live | Github
MoneyPit is a full stack crowdfunding web application, inspired by Kickstarter. Users can create and back technology based projects, earning rewards from a 3-tiered pledge system that keeps track of the total funding of the project.
React / Redux / Javascript / Node.js / Python / Flask / SQLAlchemy / JSX / CSS / Docker
Stickr
Live | Github
Stickr is a full stack web application, inspired by Flickr. This app allows users to upload and share photos of stickers they have created or found around the world. The front-end was build using React and Redux, while the back-end was built using Express.Js.
React / Redux / Javascript / Express / HTML / CSS / AWS3 / Adobe Photoshop
Covid With Strangers
Live | Github
Covid With Strangers is a single page web application inspired by Tea With Strangers. Given our current situation with the pandemic, we created an app that allows users to create virtual meetings based on different topics and meet with people virtually through zoom.
Express / Pug / Javascript / Node.js / HTML / CSS
